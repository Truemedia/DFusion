JFusion Installation Help:-

Q) What can JFusion do?

A) JFusion is capable of the following;

  1. JFusion allows you to integrate your Joomla! users with many other software(s), therefore allowing your users to register only once and then login with a single username/password across different software(s) that have been linked to each other using JFusion.

  2. Additional modules are provided to show avatars from your integrated software on the login module and to show forum posts in Joomla! itself.

Q) What can JFusion not do?

A ) JFusion does not make any changes to the integrated software, therefore if you login through your integrated software, you will not be logged into Joomla!.

Q) How does JFusion work?

A) JFusion uses the Joomla! 1.5 framework to provide user integration without any core hacks. JFusion only works with Joomla! 1.5.x and not with Joomla! 1.0.x. You can select which software is used as the master of authentication. All user registrations and password changes will need to be made through the master software. JFusion will update the software slaves with the user details from the master user table. In addition JFusion can handle multiple types of password encryption and can even create a session in the slave software(s) to automatically logon the user on all software(s).

Q) How to install and configure JFusion?

We spent quite a bit of time making the installation process as easy as possible. There is a setup wizard and extensive error checking during all steps of installation to make it easy for you. Please install and configure JFusion using the steps below;

New Installation:-

  1. Download the latest JFusion Release from JFusion.org.

  2. This package contains all required extensions i.e. Component, Modules, Plugins etc.

  3. Install the package using the Joomla!! Installation manager. This installs all the JFusion Extensions.

  4. Configure the JFusion plugins and select your master/slaves.

  5. Run the user-sync in the JFusion component to synchronize the user tables.

  6. Run the login checker to confirm whether the login behaviour is correct or not.

  7. Enable the JFusion authentication and user plugins from the JFusion Control Panel. This step also disables the default Joomla! 'auth' and 'user' plugins.

  8. Configure and activate the already installed JFusion Modules and Plugins.(optional)

Q) Does JFusion allow for Visual Integration?

A) At this point in time JFusion gives you three ways of visually integrating your software;

  1. Modify your external softwares template to match Joomla! and select the “direct link” option in the JFusion login/activity module.

  2. Show the external software in a Joomla! iframe wrapper and select the “wrapper” option in the JFusion login/activity module. You can create a menu link to your software by creating a new Joomla! menu item for the JFusion component.

  3. Full visual integration without a wrapper is supported only for phpBB and SMF. We are working on visually integrating other JFusion plugins as well using this approach.
